# Royate Hill Community Orchard

A historic community orchard in Easton, Bristol, occupying a hillside allotment site under the M32 viaduct.

## About

Royate Hill Community Orchard is a living demonstration of permaculture principles, featuring:

- Over 40 varieties of apples, plus pears, plums, cherries, and nuts
- No-dig cultivation methods
- Edible perennials and medicinal plants
- Community events, workshops, and volunteer opportunities
- A rich history dating back to the 1600s as dairy and fruit-growing land

## Website

This repository contains the static website for Royate Hill Community Orchard. The site provides information about:

- The orchard's history and story
- Permaculture practices and biodiversity
- How to get involved and volunteer
- Upcoming events and workshops
- Visiting information

## Local Development

To view the website locally:

1. Clone this repository:
   ```bash
   git clone https://github.com/QuitHub/royatehillcommunityorchard.git
   cd royatehillcommunityorchard
   ```

2. Open `index.html` in your web browser:
   ```bash
   # On macOS
   open index.html
   
   # On Linux
   xdg-open index.html
   
   # On Windows
   start index.html
   ```

Alternatively, use a local web server:

```bash
# Python 3
python -m http.server 8000

# Python 2
python -m SimpleHTTPServer 8000

# Node.js (with http-server installed)
npx http-server
```

Then visit `http://localhost:8000` in your browser.
